# .NET Project ReadMe

## Introduction
This document provides a step-by-step guide to setup, build, and run the .NET project.

## Prerequisites
1. **.NET SDK**: Ensure that you have the .NET SDK installed. You can download it from (https://dotnet.microsoft.com/download).
2. **IDE**: It is recommended to use Visual Studio for a better development experience.
3. **Database**: Ensure that the SQL Server Management Studio is installed and configured.
